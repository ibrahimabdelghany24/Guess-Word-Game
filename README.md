<h1>Guess Word Game</h1>

<p>
  Game Explanation :<br>

  <br>
  And The Tries Equal The Word Length <br>
  There Is Hints Equals Half Of The Word Length
  <br>
  You Can Add New Words And Add Some Features If You Want It's Not Perfect.
